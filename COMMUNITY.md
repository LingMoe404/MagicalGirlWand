# Community

MagicalGirlWand is maintained as a standalone CmdPal-focused fork by **泠萌404**.

This file is intentionally short. It is not a copy of the Microsoft PowerToys community roll call, and it should not imply that this repository is the official PowerToys project.

## Project community

Contributions are welcome when they match the current repository scope:

- CmdPal app behavior, UI, settings, and branding.
- CmdPal built-in extensions and extension gallery behavior.
- Extension SDK, Toolkit, and extension template improvements.
- Build, installer, validation, and standalone dependency cleanup.
- Documentation that reflects the MagicalGirlWand project rather than the full PowerToys suite.

Before opening a large change, start with an issue or discussion so the scope can be agreed first.

## Upstream acknowledgments

MagicalGirlWand is based on Microsoft PowerToys CmdPal and still retains shared code, build infrastructure, and third-party attributions from upstream PowerToys where needed.

Upstream contributors to PowerToys, CmdPal, PowerToys Run, Wox-derived launcher work, Windows App SDK integration, and related extension infrastructure deserve credit for the foundation this fork builds on. Detailed license and third-party attribution notices are kept in [NOTICE.md](NOTICE.md).
