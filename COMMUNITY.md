# 社区

MagicalGirlWand 由 **泠萌404** 维护，是一个聚焦 standalone CmdPal 的项目。

这个文件不会复制 Microsoft PowerToys 的完整社区名单，也不应让人误以为本仓库是官方 PowerToys 项目。

## 参与范围

欢迎以下方向的贡献：

- CmdPal 应用行为、UI、设置和品牌资源。
- CmdPal 内置扩展与扩展画廊。
- Extension SDK、Toolkit 和 ExtensionTemplate。
- 构建、安装器、验证脚本和 standalone 依赖清理。
- 准确反映 MagicalGirlWand 当前范围的文档。

较大的改动请先开 issue 或 discussion，把范围确认清楚后再动手。

## 文字风格

- 给人类用户看的内容中文优先。
- 人名、账号、人类 ID、项目名、库名和工具名保持原文，例如 **泠萌404**、`OpenAI Codex`、`Google Antigravity`、`Microsoft PowerToys`、`CmdPal`。
- 不要把无关 PowerToys 工具写成 MagicalGirlWand 的当前功能。

## 上游致谢

MagicalGirlWand 基于 Microsoft PowerToys CmdPal，仍保留部分上游共享代码、构建基础设施和第三方 attribution。

PowerToys、CmdPal、PowerToys Run、Wox-derived launcher work、Windows App SDK 集成和相关扩展基础设施的上游贡献者，为这个 fork 提供了基础。详细许可和第三方声明保留在 [NOTICE.md](NOTICE.md)。
